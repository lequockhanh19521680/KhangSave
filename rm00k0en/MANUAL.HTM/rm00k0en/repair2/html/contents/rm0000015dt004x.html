OUTER REAR VIEW MIRROR > INSTALLATION

Xem Phần Chuẩn Bị Kích chuột vào đây



1. INSTALL OUTER MIRROR COVER LH (w/ Cover)



  1. Install the mirror cover as shown in the illustration.




  1. Check that there is no gap between the cover and mirror body.

    GỢI Ý:
    If there is a gap between the cover and body, noise will occur during driving.




  1. Install the mirror as shown in the illustration.



2. INSTALL OUTER REAR VIEW MIRROR ASSEMBLY LH



  1. Install the mirror so that the claw attaches, and install the 3 nuts.

    Mômen xiết:
    8.0 N*m{ 82 kgf*cm , 71 ft.*lbf }

  1. w/ Power Mirror Control System:
    Connect the connector labeled A.



3. INSTALL FRONT DOOR TRIM BOARD SUB-ASSEMBLY LH



  1. Cài khớp 8 vấu để lắp gioăng cửa bên trong.

  1. Lắp tấm ốp.

    1. Cài khớp 9 kẹp để lắp ốp cửa.

  1. Lắp kẹp.



4. INSTALL FRONT ARMREST BASE PANEL UPPER LH



  1. cho Phía hành khách trước:
    Cài khớp 2 vấu để lắp công tắc kính cửa sổ vào tấm đế.




  1. cho Phía người lái:

    1. Lắp công tắc chính nâng hạ kính vào tấm cửa.

    2. Lắp 3 vít.




  1. Lắp giắc nối công tắc.

  1. Cài khớp 2 kẹp và 8 vấu để lắp tấm đế cùng với công tắc chính cửa sổ điện.

  1. Lắp tay kéo bằng một vít.



5. INSTALL FRONT DOOR LOWER FRAME BRACKET GARNISH LH



  1. Cài khớp 3 kẹp để lắp ốp trang trí.



6. INSTALL FRONT DOOR WINDOW REGULATOR HANDLE ASSEMBLY



  1. Lắp phanh hãm vào tay nắm bộ nâng hạ.

  1. Với cửa sổ đã được đóng hoàn toàn, hãy lắp tay nắm nâng hạ cửa và vòng đệm vào trục nâng hạ, như được chỉ ra trong hình vẽ.



7. PERFORM INITIALIZATION

  1. Perform initialization (see page Kích chuột vào đây).

    CHÚ Ý:
    Certain systems need to be initialized after disconnecting and reconnecting the cable from the negative (-) battery terminal.
